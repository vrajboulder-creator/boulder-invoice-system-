# Claude Code Custom Commands

## /batch
Execute multiple independent code operations in parallel for efficiency. Use for: bulk file edits, multi-file refactoring, dependency updates, batch replacements.

## /code-review-excellence
Comprehensive code review guidance for React 19, Vue 3, Rust, TypeScript, Java, Python, and C/C++. Helps catch bugs, improve code quality, and give constructive feedback.

## /code-review-graph:architecture_map
Generate architecture overview and component relationships using code-review-graph MCP tools.

## /code-review-graph:debug_issue
Debug and trace issues using code-review-graph impact analysis and flow tracking.

## /code-review-graph:onboard_developer
Generate codebase documentation and developer onboarding materials.

## /code-review-graph:pre_merge_check
Run comprehensive pre-merge checks with change detection and risk analysis.

## /code-review-graph:review_changes
Review code changes with risk scoring and detailed change analysis.

## /compact
Summarize findings and minimize unnecessary context. Use for: quick fixes, status updates, concise explanations.

## /context
Provide comprehensive context about the codebase, current file, and workspace state.

## /cost
Optimize for token efficiency. Use when token budget is a concern.

## /ui-ux-pro-max
Create distinctive, production-grade frontend interfaces with high design quality. Use for: web components, pages, landing pages, dashboards, React components, HTML/CSS layouts, styling/beautifying web UIs.

## /frontend-design
Build polished frontend code and UI design that avoids generic AI aesthetics.

## /owasp-security
Review code for security vulnerabilities, implement authentication/authorization, handle user input. Covers OWASP Top 10:2025, ASVS 5.0, and AI security.
